# Tecnologías Simples con Integración de API Google Maps

Implementé el proyecto haciendo uso de el sistema básico para generar una aplicación Web, utilizando JavasCript para el Front-End y el Back-End, base de datos no relacional con mongoDB y realizando la integración de la API de Google Maps, lo que permitió realizar la simulación de una transacción completa.

## Áreas de Impacto

### Promoción de Productos.

Implementé una plataforma en la que se puede seleccionar productos/promociones y la cantidad requerida, colocarlas en un carrito y realizar la transacción, y que el producto pueda ser entregado en la ubicación seleccionada, la plataforma permite tener un historial de las transacciones realizadas, está información permite conocer las tendencias de consumo de los clientes finales, los productos más solicitados y las ventas totales de los restaurantes.

### Finanzas

Facilita la forma en que los clientes finales pueden realizar sus pedidos y que los mismo puedan ser entregados en un tiempo prudente, esto genera un ahorro de tiempo y dinero a los clientes lo que impacta en retención y ampliación de los mismos, esto genera mayores ingresos a la plataforma.

## Tecnologías utilizadas

- mongoDB
- HTML
- JavasCript
- Google Maps API
