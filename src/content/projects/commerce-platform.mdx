# Azure

Utilice el la plataforma de Azure para implementar la solución de Arquitectura que fue seleccionada, el cual permite tener el despliegue del Front-End, el Back-End y la base de datos en un mismo grupo de recursos.

## Áreas de Impacto

### Oportunidades de Negocios

Implementé una solución de Cloud Computing en la plataforma de Azure, el cual permite generar los recursos necesarios para el despliegue y correcto funcionamiento de al aplicación de comercio electrónico, lo que deriva en mayor disponibilidad de la página que permite que los clientes puedan adquirir productos y servicios.

### Finanzas

Permite a la organización a tercerizar el servicio de Hosting de su aplicación, prescinde de una solución de infraestructura on premise, por lo cual, su inversión inicial es menor que implementar una solución en la nube, lo que permite alta disponibilidad y escalabilidad.

## Tecnologías utilizadas

- Azure
- Terraform
- SQL Server
